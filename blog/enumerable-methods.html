The map method

1/31/2016

There are many great enumerable methods available to use. The one that I am going to focus on in this post is the #map method. The #map method works by going through each value in an array, one value at a time and applying the code block to the value. The #map method will create a new array consisting of the results of the new values. One of the most important things to remember when using this method is that it returns an array. To better illustrate how this works let's look at the example below:

First we are going to take a collection of flowers(array). flowers = ['rose','tulip','daisy','sunflower','daffodil','lily','iris']

Now we are going to apply the #map method to this array to capitalize each type of flower. flowers.map {|flower| flower.capitalize}

As you can see above, the #map method is applied to the array by placing it after the array name(flowers) with a ‘.’ separating the two. The part following #map is called “the code block”. The code block has important information that #map needs to execute the task of capitalizing each type of flower. Inside the curly brackets is the element (|flower|) that #map will evaluate each time. After the element is identified, the following code “flower.capitalize” tells map what to do to the element, in this case capitalize the element.

When this example is executed the below array is returned: => ["Rose", "Tulip", "Daisy", "Sunflower", "Daffodil", "Lily", "Iris"]
